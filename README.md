Rahhal CREW

Rahhal CREW is a lightweight crisis exercise workflow chatbot that generates structured exercise outputs and exports them to DOCX.

How to run in GitHub Codespaces
1 Open this repo in Codespaces
2 In the terminal run: pip install -r requirements.txt
3 Run: streamlit run app.py
4 Open the forwarded port link

Usage
Chat with Rahhal.
When ready, type FINAL PACKAGE.
Click Export DOCX to download the output.
